# Astro Beam Recruitment Task

# Περιγραφή
Αυτό το repository περιέχει τα flowgraphs, screenshots και σημειώσεις που δημιουργήθηκαν κατά τη διάρκεια του recruitment task για την ομάδα Astro BEAM.

Το task περιλαμβάνει πειράματα με σήματα στο GNU Radio Companion (GRC), όπως:
- δημιουργία καθαρού σήματος
- προσθήκη θορύβου
- φιλτράρισμα σήματος
